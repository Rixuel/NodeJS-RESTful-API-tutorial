Node.js RESTful API tutorial

How to use?

  • Home Page: http://127.0.0.1:8081/
  • List Users: http://127.0.0.1:8081/listUsers
  • Add User: http://127.0.0.1:8081/addUser
  • Show by ID: http://127.0.0.1:8081/{id}
  • Delete User by ID: http://127.0.0.1:8081/deleteUser/{id}
Tutorial source: http://www.tutorialspoint.com/nodejs/nodejs_restful_api.htm
